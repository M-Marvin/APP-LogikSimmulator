[INSTALLATION]
The Programm can be used portable in any directory.
But optionally the file extension .lcf can be configured to point to the LogikSimulator on a fixed location.
To configure it, first copy the programm into an folder that needs no admin rights to access and then run the setup script file WITH ADMIN RIGHTS.
After the setup script has ben executed, it should be possible to open circuit files by double-clicking on them.
